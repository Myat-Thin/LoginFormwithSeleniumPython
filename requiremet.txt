#for Window 10 and above
Search bar > Microsoft store ကိုဝင်ပါ။
Microsoft Store > Search bar မှာ Python3.10 ကို Install လုပ်ပါ။
Search bar > Cmd မှာ
	-Python3 --version ကို ရိုက်ကြည့်ပါ။ Python install ဖြစ်ပြီဆိုရင် Python Version ထွက်လာပါလိမ့်မယ်။
	-Python Version ထွက်လာလျှင် pip --version ကို ရိုက်ကြည့်ပါ။ pip version ထွက်လာပါလိမ့်မယ်။
	-pip version ထွက်လာလျှင် pip install pytest ကို ရိုက်ပါ။ ပြီးလျှင် pytest လို့ ရိုက်ကြည့်ပါ။
	-Error တက်လျှင် Environment Variable Setup လုပ်ပေးရန် လိုပါတယ်။ 
	-Selenium install လုပ်ရန် > pip install selenium==4.0.0.b4 
	-Selenium webdriver install လုပ်ရန် > pip install webdriver-manager



#for Linux
sudo apt update
sudo apt install python3 (python installation) //3.10.12
python3 --version (to check python version)
sudo apt install python3-pip (pip installation)
pip --version (to check pip version)    
pip install selenium==4.0.0.b4 (selenium webdriver installation)
pip show selenium (to check selenium version)
pip install pytest (testing framework installation)
pip install webdriver-manager

#for mac
python3 --version (to check python version)
pip --version (to check pip version) 
pip install selenium==4.0.0.b4 (selenium webdriver installation)
pip show selenium (to check selenium version)
pip install pytest (testing framework installation)
pip install webdriver-manager
